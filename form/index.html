Оболочка формы <form></form>

Это формы поиска, регистрации и авторизацции. Позволяют взаимодействовать с пользователем. Все данные передаются на сервер.
Атрибуты:
  • action - куда полетят данные (сервер);
  • method - метод передачи данных (get - видны в адресной строке(подходит для малоко количества данных), post - передает данные скрытно);
  • enctype="multipart/form-data"
<form action="#" method="" enctype="multipart/form-data"></form>

Поле ввода <input>

input type="text" - метод приема данных от пользователя. name - имя, value - значение
  • placeholder - серый цвет подсказка
  • disabled - блокирует поле, не отправляет на сервер
  • tabindex - порядок перехода по ячейкам TAB

Выпадающий список <select></select> + <option></option>

  • option - пункт выпадающего списка

Выпадающий список select имеет атрибут multiple (выбор несокльких вариантов с нажатым ctrl)

Кнопки <button></button>

  • submit - отправиться вся форма
  • reset - очистится вся форма
