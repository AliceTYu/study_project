HTML теги. DIV, HEADER, FOOTER, NAV

HTML теги - это кирпичики, из которых строится надежный и крепкий каркас сайта.
Строительные теги предназначены для создания структуры/каркаса.
Строительные теги - блочные, каждый блок выводится с новой строки.
Структура: <имя тега> содержание тега </имя тега>
<header> верхняя часть сайта (шапка) </header>
<nav> навигация или меню </nav>
<footer> нижняя часть сайта (подвал) </footer>

Текстовый абзац <p></p>

Блочный тег <p></p>. Блочный тег занимает всю строку экрана, независимо от того скок занимает само содержимое

Выделение куска текста - <span></span>

Строчный тег <span></span> - пишется в строку один за другим. Строчный тег - его ширина зависит от количества содержимого внутри него.

содержание тега span №1 содержание тега span №2 Lorem ipsum dolor sit amet consectetur adipisicing elit. Sunt, qui repellendus tempore, officiis quaerat laudantium eligendi libero perferendis dicta hic delectus? Vero culpa deserunt impedit velit ipsa vel facilis aut.

В строчный тег НЕЛЬЗЯ записать блочный тег. Но внутрь блочных тегов МОЖЕМ помещать строчные.

Ссылка <a></a>

Тег а, ссылка на тот или иной документ, сайт или файл. Строчный.
Атрибуты: href - ссылка на другую страницу или сайт; target="_blank" - содержимое откроется в другом окне
Goggle

Изображение <img>

Вставка изображений. Строчный.
Атрибуты: src - путь к картинке. alt - альтернативный текст, если картинка не загрузится

Маркированный список <ul></ul> + <li></li>

  • Сначала объявляется ul
  • Внутри каждый пункт списка обрамляется в li
  • содержание тега 3

Нумерованнный список <ol></ol> + <li></li>

  1. Сначала объявляется ol
  2. Внутри каждый пункт списка обрамляется в li
  3. содержание тега 3
