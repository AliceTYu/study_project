CSS

Каскадная таблица стилей - отвечает за то, как будет выглядеть страница.
Структура:
  • div { //селектор
  • font-size: 20px; //параметр. font-size - свойство; 20px - значение
  • }
Подключение:
  • <link rel="stylesheet" href="/css/style.css">
Обращение:
  • по классам class="name"
У каждого тега есть стили по умолчанию, и каждый браузер отображает их по своему. Поэтому их надо обнулять и самому задавать нужные стили.
